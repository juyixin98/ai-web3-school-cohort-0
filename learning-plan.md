# 个人学习计划

> 最后更新：2026-05-18

## 目标

以 **Hackathon 项目** 为最终交付目标，在 2h+/天的投入下，补齐 AI × Web3 交叉知识，形成可演示的原型。

## 学习路径

### 第一阶段：基础补齐（前 2 周）

根据学员画像，AI 和 Web3 两端都已有基础，重点放在 **AI × Web3 Bridge**：

- [ ] [LLM](https://aiweb3.school/zh/handbook/ai/llm/) — 快速过一遍，确认理解大模型能力边界
- [ ] [Agent](https://aiweb3.school/zh/handbook/ai/agent/) — 深入理解 Agent 架构与工具调用
- [ ] [Wallet](https://aiweb3.school/zh/handbook/web3/wallet/) + [Account Abstraction](https://aiweb3.school/zh/handbook/web3/account-abstraction/) — 理解智能账户与 Agent 权限
- [ ] [Smart Contract](https://aiweb3.school/zh/handbook/web3/smart-contract/) — 巩固合约开发基础

### 第二阶段：交叉核心（第 3–4 周）

- [ ] [Chain-aware Context](https://aiweb3.school/zh/handbook/bridge/chain-aware-context/) — 链上状态如何进入 Agent 上下文
- [ ] [Web3 Tool Use](https://aiweb3.school/zh/handbook/bridge/web3-tool-use/) — Agent 如何调用链上工具
- [ ] [Agent Workflow](https://aiweb3.school/zh/handbook/bridge/agent-workflow/) — 自动化与 human-in-the-loop 边界
- [ ] [Agent Wallet](https://aiweb3.school/zh/handbook/bridge/agent-wallet/) — Agent 拿什么权限、如何限制
- [ ] [Machine Payment](https://aiweb3.school/zh/handbook/bridge/machine-payment/) — 机器间支付与结算

### 第三阶段：Hackathon 准备（第 5–6 周）

- [ ] 选定 Hackathon 赛道，深入阅读 [前沿探索](https://aiweb3.school/zh/handbook/tracks/agentic-commerce/) 对应章节
- [ ] 搭建项目原型（实验代码放入 `experiments/`）
- [ ] 方案文档（放入 `hackathon/`）
- [ ] 演示准备

## 每周节奏

| 日期 | 重点 |
|---|---|
| 周一–周五 | 每日 2h：阅读 Handbook + 实验/笔记 |
| 周六 | 整理本周笔记，提交 feedback |
| 周日 | 轻量复习或休息 |

## 打卡方式
