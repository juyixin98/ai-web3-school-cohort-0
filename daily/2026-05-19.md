## 关键收获

1. Agent 查询链上数据本质上就是 HTTP JSON-RPC 请求，和调用任何 Web2 API 一样，`eth_blockNumber` / `eth_getBalance` / `eth_gasPrice` 就是 Agent 的 tool
2. 链上数据全以 hex 返回（Wei 单位），Agent 需要额外做格式转换才能给人类/模型阅读理解
3. RPC 节点是 Agent 进入链上世界的「网关」——选对节点（公共 vs 私有 vs 付费）影响可用性和隐私
4. 只读调用（eth_call / eth_getBalance）是安全的；写操作（eth_sendTransaction）需要私钥签名，必须走权限控制
