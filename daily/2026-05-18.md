## 关键收获

1. Agent 从「回答问题」升级到「多步执行」的核心是 Tool Use 循环：模型推理 → 选择工具 → 执行 → 读取结果 → 继续推理，直到任务完成
2. Chain-aware Context 的本质是把链上数据（区块高度、gas、交易历史、合约事件）结构化地注入 Agent 上下文，让模型能感知链上世界状态
3. Web3 Tool Use 让 Agent 可以直接调用 RPC（eth_call、eth_sendTransaction 等）、钱包签名、合约读写，关键挑战在于权限边界和安全执行
