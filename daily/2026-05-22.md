## 关键概念预习

| 概念 | 一句话 | 难度 |
|---|---|---|
| **Stablecoin Payment** | 稳定币适合机器支付：价格稳定、可编程、可验证 | 初级 |
| **Budget** | 预算先于执行——按时间/任务/服务方/币种限制，不设总预算就要设频率上限 | 初级 |
| **Quote** | 可执行报价：价格、币种、有效期、交付条件、退款条件、quote id | 中级 |
| **Payment Intent** | 付款意图≠已结算——绑定任务/金额/收款方/有效期，给后续操作提供上下文 | 中级 |
| **x402** | HTTP 402 = 付费才能访问资源，Agent 像处理 401 一样处理 402 | 中级 |
| **MPP** | 机器间支付协议：服务发现→报价→授权→结算→收据 | 中级 |
| **Subscription** | 订阅需要撤销能力：用户要能看到何时续费、上限、能否涨价 | 中级 |
| **Micropayment** | 高频小额不适合每笔上链，适合 L2/批量结算/预付余额 | 高级 |
