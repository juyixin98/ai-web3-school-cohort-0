### 关键决策

- **数据读取用 mainnet**：免费 RPC，数据真实有意义，适合 demo 展示
- **交易执行留 Sepolia**：安全测试，不涉及真实资金
- **DEX 用 V2 pair 地址计算**：通过 factory + token0/token1 + init code hash → CREATE2 地址，不需要手动维护 pair 列表
